# TradieMind AI - Project Roadmap

## Phase 1: MVP (Months 1-3)
**Goal:** Core voice quoting functionality

### Month 1: Foundation
- [ ] Project setup (repo, CI/CD, infrastructure)
- [ ] Database schema design & implementation
- [ ] Authentication system
- [ ] Basic iOS/Android app shell

### Month 2: Voice Core
- [ ] Voice recording & upload
- [ ] Whisper integration (speech-to-text)
- [ ] Intent classification (GPT-4)
- [ ] Quote generation engine
- [ ] PDF generation

### Month 3: Integration
- [ ] Xero/MYOB connection
- [ ] Email sending (quotes)

## Risk Mitigation

| Risk | Mitigation |
|------|------------|
| AI accuracy | Start with narrow use cases, human-in-the-loop |
| Voice quality | Offline mode, manual override always available |
| Integration failures | Graceful degradation, manual entry fallback |
| Competition | Speed to market, Australian focus, voice-first |
| Funding | Revenue from Month 6, conservative burn rate |
